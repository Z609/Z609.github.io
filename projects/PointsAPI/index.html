Z609 PointsAPI

PointsAPI is a small points system. It isn't the most efficient thing on the planet, but it's one of the best points plugins out there. It was originally intended to be a custom plugin for a smaller server that requested that I code a plugin for them. I ended up releasing it because they said they wouldn't mind if I released it. Please note that upon decompilation of the code, you will notice a bunch of inefficient code. I will re-code it sometime in the future to be more efficient, but for now, I don't have time. Regardless, here is the documentation, which has been on many other sites before but has been lost because of constant website switchovers.

Set player's points

You can use PointsAPI#setPoints(); to set a player's points. You are required two arguments. The first argument must be a player representation. The first argument can be one of the following:

  • org.bukkit.entity.Player - An online Bukkit player
  • java.util.UUID - UUID for a player (org.bukkit.entity.Player#getUniqueId())
  • java.lang.String - Player's UUID (in a String java.util.UUID#toString()) to be affected.

The second argument must be an integer. This will set the amount of points right away.

Example: 
@EventHandler
public void onInteract(PlayerInteractEvent event){ // It's a player interact event!
	if(event.getAction() == Action.RIGHT_CLICK_AIR || event.getAction() == Action.RIGHT_CLICK_BLOCK){ // Only when right clicking
		Player player = event.getPlayer(); // Fetch the player for usage
		if(player.getItemInHand() != null && player.getItemInHand().getType() == Material.POTATO_ITEM){ // If the item in their hand is a potato
			player.sendMessage("Thank you for clicking my potato :)"); // Send them a cute message
			player.playSound(player.getLocation(), Sound.CLICK, 1.0F, 1.0F); // Play them a fancy sound
			PointsAPI.setPoints(player, 300); // Set their balance to 300
		}
	}
}

Add/take player's points

You can use PointsAPI#addPoints(); or PointsAPI#takePoints(); to add a deduct a player's balance. You are required two arguments. The first argument must be a player representation. The first argument can be one of the following:

  • org.bukkit.entity.Player - An online Bukkit player
  • java.util.UUID - UUID for a player (org.bukkit.entity.Player#getUniqueId())
  • java.lang.String - Player's UUID (in a String java.util.UUID#toString()) to be affected.

The second argument must be an integer. This will add or deduct the balance depending on the method used.

Example adding points: 
@EventHandler
public void onInteract(PlayerInteractEvent event){ // It's a player interact event!
	if(event.getAction() == Action.RIGHT_CLICK_AIR || event.getAction() == Action.RIGHT_CLICK_BLOCK){ // Only when right clicking
		Player player = event.getPlayer(); // Fetch the player for usage
		if(player.getItemInHand() != null && player.getItemInHand().getType() == Material.POTATO_ITEM){ // If the item in their hand is a potato
			player.sendMessage("Thank you for clicking my potato :)"); // Send them a cute message
			player.playSound(player.getLocation(), Sound.CLICK, 1.0F, 1.0F); // Play them a fancy sound
			PointsAPI.addPoints(player, 300); // Increase their balance by 300
		}
	}
}


Example deducting points: 
@EventHandler
public void onInteract(PlayerInteractEvent event){ // It's a player interact event!
	if(event.getAction() == Action.RIGHT_CLICK_AIR || event.getAction() == Action.RIGHT_CLICK_BLOCK){ // Only when right clicking
		Player player = event.getPlayer(); // Fetch the player for usage
		if(player.getItemInHand() != null && player.getItemInHand().getType() == Material.POTATO_ITEM){ // If the item in their hand is a potato
			player.sendMessage("Don't click my potato!"); // Send them a mean message
			player.playSound(player.getLocation(), Sound.CLICK, 1.0F, 1.0F); // Play them a fancy sound
			PointsAPI.takePoints(player, 300); // Decrease their balance by 300
		}
	}
}
